# Stewart's Home Registry

A Vercel-ready static website for Stewart's Home Registry.

## Edit the registry

Update `registryItems` in `app.js` to change gift names, rooms, prices, priorities, and notes.

Update `mailingAddress` in `app.js` before publishing the site publicly.

## Deploy on Vercel

1. Push this repository to GitHub.
2. In Vercel, choose **Add New Project**.
3. Import `woftonss/stewarts-home-registry`.
4. Keep the framework preset as **Other** and deploy.

No build command or output directory is required.
